# Blockpose
A straightforward blockchain indexer for Ethereum Virtual Machine (EVM) networks. Simplifies the process of writing transformation code in JavaScript and streaming data to various destinations.

# Installation

#### With npm
```
npm i -g @keltlabs/blockpose
```
#### With yarn
```
yarn global add @keltlabs/blockpose
```

### Notes
<b style="color:orange">Currently, our support for the command-line interface (CLI) is available exclusively for the Mac OS on ARM64/AMD64 and Linux on X64 architectures.</b>

# Guide
## How to create new indexer?
### Create new `config.json` as below
```
{
  "rpc_url": "https://bscrpc.com",
  "block_time": 3,
  "function": "main.js"
}
```

### Run the indexer
Place the configuration and transformation function files in the same directory and execute the following command to observe the results:
```
blockpose
```

Alternatively, you can specify the configuration and function files with the following command:
```
blockpose -c path_to_config.json -f path_to_function.js
```
